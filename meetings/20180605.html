Steady State Surface Laplacian

Sage found that the differenetiation matrix approximating the surface laplacian was singular and thought this to be an error. Grady demonstrated that it is correct. Consider $\Delta_\SS u = 0$. Any constant function $u(\vec{x}) = c$ is a solution to this, thus the system must be underdetermined. In order to have a meaningful solution to a steady state problem without boundary conditions more information is needed.

Grady suggested trying to fix the temperature at a point (effectively creating a boundary condition). Alternatively he suggested adding the constraint that $\int_\SS u = 0$, that the average value is zero.
