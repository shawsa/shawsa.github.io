RBF-Interpolation

Fri Sep 21 23:13:59 2018

Given a set of distinct points $\{\vec{x}_i\}_{i=1}^n \subseteq D$ and function values $\{f_i\}_{i=1}^n \subseteq \RR$ (or $\CC$) at those points an interpolant $s : D \to \RR$ is a function such that $s(\vec{x}_i) = f_i$ for $i=1, 2, ..., n$. This article discusses RBF interpolation and compares it to polynomial interpolation.

Numpy Efficiency

Wed Sep 19 00:16:56 2018

One of the most attractive features of Python is that list comprehensions allow the expression of loops in a clear human readable format that avoids indices. In the context of scientific computing, Numpy is often used to merge the speed of C and Fortran libraries with the clear presentation of Python. If one naively wraps Python list comprehensions in Numpy arrays, they lose much of the efficiency provided by Numpy and may be less efficient than pure Python! This article discusses efficient use of Numpy and best practices.

Laplacian

Wed Sep 19 00:16:48 2018

The laplacian is a second order differential operator. When applied to the function $u$ it is denoted $\nabla^2u$ or $\Delta u$. It is the sum of the second partial derivatives with respect to each dimension. It's notation can be understood by noting that the sum of second partials is the divergence of the gradient $\nabla^2u = (\nabla \cdot \nabla) u$.

Experiment 003 RBF Independent Error

Sat Sep 1 17:46:26 2018

It has been obsereved that when using iterated differentiation to solve the problem $\Delta_\SS u = -20Y_{4,-3}$ on the unit sphere, if the degree of the basis terms is 5 (i.e. one higher than the degree of the forcing function) then the error is independent of the choice of RBF.
