RBF-Interpolation

Thu May 31 12:22:16 2018

Given a set of distinct points $\{\vec{x}_i\}_{i=1}^n \subseteq D$ and function values $\{f_i\}_{i=1}^n \subseteq \RR$ (or $\CC$) at those points an interpolant $s : D \to \RR$ is a function such that $s(\vec{x}_i) = f_i$ for $i=1, 2, ..., n$. This article discusses RBF interpolation and compares it to polynomial interpolation.

Laplacian

Sun May 27 19:58:42 2018

The laplacian is a second order differential operator. When applied to the function $u$ it is denoted $\nabla^2u$ or $\Delta u$. It is the sum of the second partial derivatives with respect to each dimension. It's notation can be understood by noting that the sum of second partials is the divergence of the gradient $\nabla^2u = (\nabla \cdot \nabla) u$.
